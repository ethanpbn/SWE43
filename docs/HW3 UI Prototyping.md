## Home Page

- Home page where users can see all their options in the app and also see the locations of nearby friends and cafes
- Users can scroll through the list at the bottom to see their friends' current activities
- Friends will have a distance and an activity tag to show if they are at a certain location — friends can turn this off in settings
- Consistent color theming and minimalist design so users are not easily overwhelmed when entering the app
- Users always have access to the search bar above; they can use the tabs below or the menu on the top right to access more options
- The bottom tab will have the icon enlarged to indicate which page the user is on — the bottom panel is always present so users can click one to back out from anywhere
- Map also functions like a normal map app
- Filter button above to narrow search results

---

## Cafe Searching

- Users can go to the cafe tab as a shortcut to see nearby cafes
- Users can still search above but the scroll menu below shows local cafes instead of friends
- Same layout as the home page to maintain consistency throughout the app

---

## Profile Page

- The settings tab which houses all other features that are not used as often
- Users can customize their profiles, look at past events/favorites, manage who they are blocking, and more
- Consistent color scheme, layout, and buttons to match theming

**Buttons available on the profile page:**

| Button | Function |
|---|---|
| Favorites | View saved cafes |
| My Events | View upcoming and past events |
| History | View visit history |
| Blocks | Manage blocked users |
| Location Settings | Toggle location sharing |
| Language | Change app language |
| Switch to Business | Switch to business account mode |
| Feedback | Submit feedback |
| Terms of Service | View terms |
| Log Out | Sign out of account |
